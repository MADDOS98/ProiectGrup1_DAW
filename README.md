
# Proiect: Gestionarea Activităților

Acest proiect este o aplicație web pentru gestionarea activităților, realizată în cadrul lucrului de grup la disciplina Dezvoltarea Aplicatiilor Web.

## Descriere

Aplicația permite:
- Adăugarea, vizualizarea, editarea și ștergerea activităților
- Organizarea activităților pe utilizatori
- Interfață modernă cu Bootstrap
- Autentificare și gestionare utilizatori

## Tehnologii folosite
- Laravel (PHP)
- Bootstrap (CSS)
- SQLite

## Instalare și rulare
1. Clonați proiectul
2. Rulați `composer install` și `npm install`
3. Configurați baza de date în `.env`
4. Rulați migrările cu `php artisan migrate`
5. Porniți serverul cu `php artisan serve`
